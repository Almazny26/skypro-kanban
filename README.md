# Skypro Kanban - Приложение для управления задачами

Современное веб-приложение для управления задачами в формате Kanban доски, разработанное на React с использованием Vite.

## Основные возможности

### Управление задачами

- **Создание задач** - добавление новых задач с названием, описанием, категорией и датой
- **Редактирование задач** - изменение всех параметров существующих задач
- **Удаление задач** - удаление задач с подтверждением
- **Просмотр задач** - детальный просмотр информации о задаче в модальном окне
- **Drag-and-Drop** - перетаскивание карточек между колонками для изменения статуса задачи

### Организация задач

- **Kanban доска** - задачи организованы по статусам:
  - Без статуса
  - Нужно сделать
  - В работе
  - Тестирование
  - Готово
- **Drag-and-Drop** - интуитивное перетаскивание карточек между колонками:
  - Захватите карточку и перетащите её в другую колонку
  - Статус задачи автоматически обновится
  - Изменения сохраняются на сервере
  - Визуальная обратная связь при перетаскивании
- **Категории задач** - задачи можно классифицировать по категориям:
  - Web Design
  - Research
  - Copywriting

### Поиск и фильтрация

- **Поиск задач** - быстрый поиск задач по названию в реальном времени
- **Фильтрация по категориям** - фильтрация задач по выбранной категории
- **Очистка фильтров** - быстрое сброс всех фильтров одной кнопкой

### Drag-and-Drop

- **Перетаскивание карточек** - интуитивное изменение статуса задач:
  - Захватите карточку мышью и перетащите в нужную колонку
  - Статус задачи автоматически обновится
  - Изменения сохраняются на сервере
  - Визуальная обратная связь с полупрозрачностью при перетаскивании
  - Поддержка клавиатурной навигации для доступности

### Темная тема

- **Смена темы** - переключение между светлой и темной темой
- **Сохранение темы** - выбранная тема сохраняется в localStorage
- **Адаптивные цвета** - все компоненты автоматически адаптируются под выбранную тему

### Уведомления

- **Toast-уведомления** - красивые всплывающие уведомления для всех операций:
  - Успешное создание задачи
  - Успешное обновление задачи
  - Успешное удаление задачи
  - Ошибки при выполнении операций
- **Автоматическое закрытие** - уведомления автоматически исчезают через 3 секунды

### Аутентификация

- **Регистрация** - создание нового аккаунта
- **Авторизация** - вход в существующий аккаунт
- **Выход** - безопасный выход из аккаунта
- **Защищенные маршруты** - доступ к приложению только для авторизованных пользователей

### Обработка ошибок

- **Валидация полей** - проверка на пустые поля и поля с пробелами
- **Обработка сетевых ошибок** - информативные сообщения при проблемах с подключением
- **Обработка ошибок сервера** - отображение детальных сообщений об ошибках от API

### UI/UX

- **Адаптивный дизайн** - корректное отображение на всех устройствах
- **Анимация загрузки** - красивый спиннер при загрузке данных
- **Плавные переходы** - плавные анимации при смене темы и переходов
- **Страница 404** - стилизованная страница для несуществующих маршрутов

- **@dnd-kit/utilities** - утилиты для dnd-kit
- **date-fns 4.1.0** - работа с датами

## Установка и запуск

### Предварительные требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр продакшен сборки

```bash
npm run preview
```

## Дополнительные функции

### Drag-and-Drop

Перетаскивание карточек между колонками - это основной способ изменения статуса задач:

1. Захватите карточку задачи мышью (курсор изменится на "grab")
2. Перетащите карточку в нужную колонку (статус)
3. Отпустите кнопку мыши - статус автоматически обновится
4. Вы получите уведомление об успешном обновлении
5. При ошибке изменения автоматически откатятся

**Совет:** Карточки можно перетаскивать только между колонками статусов. Перетаскивание на другие карточки игнорируется.

### Смена темы

Тема приложения переключается через чекбокс в меню пользователя (правый верхний угол). Выбранная тема автоматически сохраняется и восстанавливается при следующем посещении.

### Поиск и фильтрация

- Введите текст в поле поиска для фильтрации задач по названию
- Выберите категорию из выпадающего списка для фильтрации по категории
- Нажмите кнопку "Очистить" для сброса всех фильтров

### Уведомления

Все операции с задачами сопровождаются уведомлениями:

- Зеленые уведомления - успешные операции
- Красные уведомления - ошибки
